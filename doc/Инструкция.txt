Требуемое ПО 

- JDK 7
- СУБД PostgreSQL
- pgAdmin (включена в состав PostgreSQL)
- IDE NetBeans 8 


Развертывание базы данных:

1. Скачать с официального сайта последнюю стабильную версию PostgreSQL 9.4.5 - http://www.postgresql.org/download/ 
2. Установить PostgreSQL (можно использовать настройки предлагаемые по-умолчанию), установить пароль для пользователя "postgres" - "Iqex1234".
3. Запустить pgAdmin и подключиться к базе данных (см. рисунок 1)
4. Создайте базу данных с именем "Cities", владелец "postgres (см. 2-й рисунок)
5. После создания выделите имя базы и нажмите кнопку "Выполнить пользовательские SQL запросы"
6. В открывшееся окно скопировать содержимое файла Cities\install\dbcreate-populate-tables.sql и нажать кнопку "Выполнить запрос" (треуголньик вверху) (см. 3-й рисунок)


Развертывание проекта:

1. Разорхивировать проект Cities.zip
2. Добавить папку Cities в проекты NetBeans
3. Запустить проект (Выполнить -> Запустить проект)

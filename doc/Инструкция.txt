Требуемое ПО 

- JDK 7
- СУБД PostgreSQL
- pgAdmin (необязательно, только если захотите поиграться с запросами)
- IDE NetBeans 8 

Инструкция по инсталляции БД

1. Скачать с официального сайта последнюю стабильную версию PostgreSQL 9.4.5 - http://www.postgresql.org/download/ 
2. Установить PostgreSQL (можно использовать настройки предлагаемые по-умолчанию).
3. Скачать pgAdmin - http://www.pgadmin.org/download/
4. Установить pgAdmin (так же, по-умолчанию настройки)
5. Запустить pgAdmin и подключиться к базе данных (настройки по-умолчанию см. в картинке 1.Подключение к БД.png, пароль тот, который указали при установки PostgreSQL)
6. Создайте базу данных с именем "citizens", владелец "postgres (см. картинки 2 и 3)
7. После создания выделите имя базы и нажмите кнопку "Выполнить пользовательские SQL запросы" (см. картинку 4) 
8. В открывшееся окно скопировать содержимое файла create-citizens.sql и нажать кнопку "Выполнить запрос" (треуголньик вверху) (см. картинку 5)
9. После создания базы выполнять SELECT запросы из файла select-citizen.sql (см. картинку 6)

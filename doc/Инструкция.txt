Требуемое ПО 

- JDK 7
- СУБД PostgreSQL
- pgAdmin (включена в состав PostgreSQL)
- IDE NetBeans 8 

Инструкция по инсталляции БД

1. Скачать с официального сайта последнюю стабильную версию PostgreSQL 9.4.5 - http://www.postgresql.org/download/ 
2. Установить PostgreSQL (можно использовать настройки предлагаемые по-умолчанию).
3. Запустить pgAdmin и подключиться к базе данных
4. Создайте базу данных с именем "cities", владелец "postgres
5. После создания выделите имя базы и нажмите кнопку "Выполнить пользовательские SQL запросы"
6. В открывшееся окно скопировать содержимое файла create-populate-tables.sql и нажать кнопку "Выполнить запрос" (треуголньик вверху)
7. После создания базы выполнять SELECT запросы из файла make-select.sql
